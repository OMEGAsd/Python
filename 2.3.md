```py
if a > b:
    print(a)

```

# Not
not(X) يعكس قيمة المتغير
مثال :

```py
x = true
print(x)
not(x)
print(X)
```
```
True
False
```

# Or
يجب تحقق شرط واحد بسبب وجود or بمعنى او

if الشرط or الشرط:

    مجموعه الاوامر

else:

    مجموعه الاوامر

مثال : 
```py
x = true
y = false

if x or y :
    print(true)
else:
    print(false)
```
```
True
```
بسبب وجود قيمة واحدة true
# And
يجب تحقق جيع الاوامر بسبب وجود and بمعنى وَ

if الشرط and الشرط: 

    مجموعه الاوامر

else:

    مجموعه الاوامر

مثال :
```py
x = true
y = false

if x and y :
    print(true)
else:
    print(false)
```
```
False
```
بسبب عدم وجود القيمتين true
